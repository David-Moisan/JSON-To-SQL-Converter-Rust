# Rust JSON to SQL Converter GUI

## How to install the project

*To install the project, make sure you have Rust on your desktop: https://www.rust-lang.org/tools/install*

make this cmd :
`cargo build`

## How to run this project

`cd json_to_sql`

`make run`
